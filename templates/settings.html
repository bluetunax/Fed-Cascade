📡 Live Data Intelligence Source

Fed Cascade is an Open-Source Intelligence (OSINT) engine that connects directly to the official USAspending.gov API managed by the U.S. Department of the Treasury.

Target Isolation

The engine's payload specifically explicitly limits queries to the Agency for International Development (USAID). It drops domestic data to exclusively target foreign `ISO-3` locations.

Pipeline Segmentation

Data is rigorously segmented. The Macro Layer isolates definitive contracts and purchase orders, while the Micro Layer aggregates project grants, loans, and other financial assistance.

ENDPOINT: /api/v2/search/spending_by_award/ | FORMAT: JSON | HTTPS ENCRYPTED

📤 Collaboration & Export

Extract your entire local SQLite intelligence cache as a portable JSON file. This allows you to securely share audited 10-year ledgers, anomaly logs, and project dictionaries with colleagues without risking database corruption.

📥 Import & Merge Data

Upload a Fed Cascade JSON export file. The engine will safely ingest the new data, map new primary keys to prevent ID collisions, and seamlessly merge the foreign data into your existing local cache.
