This hub acts as a Git-style repository manager for your financial datasets. Because USAspending data changes retroactively, you can maintain multiple offline "branches" (snapshots) for the same target zone/year. You can click Merge Into... to combine an older branch with the active HEAD branch to consolidate missing offline data.

Watchdog Mode: Click Set Base on a snapshot to designate it as your anchor. All future comparisons and audits will be measured against it to detect retroactive volatility.
